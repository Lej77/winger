Keyboard shortcut

The keyboard shortcut to open the pop-up panel is currently [none]. (Default: Alt+W)

If desired, it can be changed in Firefox's centralized extension shortcuts page: Click the ⚙️ button above this page, click Manage Extension Shortcuts, and look for the Winger shortcut.

Private windows

Private windows are currently visible to Winger (recommended).

Private windows are currently not visible to Winger. It is recommended to allow Winger access to private windows.

To change access to private windows, go to the Details tab at the top this page and choose an option for Run in Private Windows.
