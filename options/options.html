Keyboard shortcut

The keyboard shortcut to open the pop-up panel can be changed in Firefox's centralized extension shortcuts page.

Click the ⚙️ button above this page, click Manage Extension Shortcuts and look for the WinMan shortcut.

Private windows

It is recommended to allow WinMan to work in private windows; otherwise WinMan will simply not see them.

If you have not done so, go to the Details tab at the top this page, then for Run in Private Windows choose Allow.
