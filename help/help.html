WinMan (Window Manager)
Quick Guide

The WinMan Popup

Click on the WinMan toolbar button to open the popup.

Every window is represented by a row, showing its name and how many tabs it contains.

The top row represents the currently focused window (current window).

Below it is the WinMan omnibox.

Below the omnibox is a list of other windows, sorted by most recently focused (seen). So the first row below the omnibox represents the previously focused window.

Below the list is the status bar showing the total number of tabs and windows open.

And below that are supportive function buttons.

Basic Actions

• Switch Window

Click on a row to switch to the window.

• Send Tab

The send button "sends" the selected tab(s) to the corresponding window while staying on the current window.

Holding Ctrl while doing a switch changes it into a send. Thus, you can also Ctrl-click a row to send tabs to that window.

• Bring Tab

The bring button switches to the corresponding window while "bringing" the selected tab(s) along. Essentially switch and send combined.

Holding Shift while doing a switch changes it into a bring. Thus, you can also Shift-click a row to bring tabs to that window.

Private window behaviour: Tabs cannot conventionally be moved between normal and private windows, so send and bring actions will reopen tabs in the destination window instead. (Page state, such as filled form data and scroll position, will be lost.)

Edit Mode

The edit button lets you name/rename the corresponding window.

When invoked the popup enters Edit mode, during which you can type in a name and use the following keys:

  • Enter to save the currently edited name and exit Edit mode.
  • Up or Down to save the currently edited name and edit another window name.
  • Esc to close the popup, exiting Edit mode and cancelling any changes to the currently edited name.

Names must be unique and may not begin with a forward slash ("/").

Note: WinMan assigns every window a default name (Window 1, Window 2 and so on), visible when a window has no user-given name. Unlike given names, default names are temporary and are not guaranteed to persist across browser sessions.

The Omnibox

Multipurpose input field that allows execution of any action via keyboard.

Primary omnibox powers:

  • Start typing in a window name (or part of it) to filter the list of other windows.
  • Press Enter to switch to the window represented by the first row below the omnibox.

And remember that holding Ctrl or Shift while doing a switch changes it to a send or bring respectively.

Thus, to switch | send tabs | bring tabs to a window via keyboard:

  1. Alt+W (the default keyboard shortcut*) to open the popup.
  2. If the window row is already immediately below the omnibox, skip this step. Otherwise, start typing its name until it is.
  3. Enter | Ctrl+Enter | Shift+Enter.

*If it has changed, hover your pointer over the WinMan button to check what it is.

Slash commands

Other actions can be performed by typing one of these slash commands, followed by Enter:

  • /help, /h or just / – Open this help page
  • /edit or /e – Enter Edit mode to rename windows
  • /settings or /s – Open WinMan settings
